# 项目联调前-checkList

##开发前checkList

- [] 是否明确了项目任务分工（如有合作）
- [] 是否明确了本期要完成的所有功能点
- [] 是否比对了

##视觉验收

- [] 在前端开发完成后，务必邀本项目UI设计师进行视觉验收。
- [] 在视觉验收时，要确认所有样式、文案相关的问题，从简单的标题样式、提示文案，到整个项目的整体页面结构，过渡动画，都要有UI确认，特别是在移动端，更加要注意视觉验收。
- [] 完成视觉验收后，也能避免很多不必要的视觉和文案相关的BUG。
